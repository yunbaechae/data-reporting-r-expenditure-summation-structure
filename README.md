# Data Reporting in R: Expenditure Summation and Structure Visualisation

STAT1378 Coding and Communication in Statistics Assignment 1 (2021)

This assignment focused on R-based data reporting and structure inspection. Tasks included CSV loading, total/conditional aggregation, and key metric identification with clear outputs.

## Key Tasks
- Imported and inspected expenditure data structure  
- Computed total sums and filtered aggregates (e.g., March cloudy weather)  
- Identified highest expenditure month  

## Skills Demonstrated
- R programming: data frames, summarization, filtering  
- Data structure inspection and reporting  
- Reproducible aggregation workflows
